# Sistema-de-cadastro-de-fam-lias-para-o-grupo-Vicentinos
Sistema de Cadastro de Famílias para o Grupo Vicentinos da Paróquia São Pio X. O objetivo é apoiar a gestão das informações das famílias atendidas pela instituição por meio da digitalização e centralização dos registros cadastrais. A solução foi o desenvolvimento de um sistema desktop simples e intuitivo.
