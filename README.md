# Sistema-de-cadastro-de-fam-lias-para-o-grupo-Vicentinos
SysVi – Sistema de Cadastro de Famílias para o Grupo Vicentinos da Paróquia São Pio X. O objetivo é apoiar a gestão das informações das famílias atendidas pela instituição por meio da digitalização e centralização dos registros cadastrais. A solução foi o desenvolvimento de um sistema desktop simples e intuitivo, capaz de armazenar, consultar, editar e excluir informações cadastrais de forma organizada. A proposta é substituir gradualmente os registros manuais, proporcionando maior praticidade no gerenciamento das informações e contribuindo para a melhoria dos processos administrativos da instituição. O desenvolvimento do SysVi foi realizado através do projeto extensão do Curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas do Instituto Federal de Educação, Ciência e Tecnologia de São Paulo (IFSP), possibilitando a aplicação dos conhecimentos acadêmicos na resolução de uma demanda real da comunidade.
